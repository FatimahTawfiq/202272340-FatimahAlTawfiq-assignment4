# Overview
This assignment is creation of a personal portfolio website using HTML, CSS, and javaScrept. 
The portfolio consist of five sections with a top Bar:
#### 1-Top Bar 
This part contains a stick navigation bar to move to specific section and theme icon to change the theme to light or dark mode.

#### 2-Introduction
This section contains a greating message to the user based on the day time, my name, hook, inspiration quote.

#### 3-About Me
This section contains a breaf information about me, my education, experianses and aims, and time remaining to graduate.

#### 4-Skills
This section contains the technical skills that I gained during my educational period until now.

#### 5-Projects
This section is about my personal projects with a small descripthin and image of each one.
Each project has its oun card with a gap between each card. The user can see all projects or use the bar to filter projects based on the fields and sort from newest to oldest or from oldest to newest.

#### 6-Contact Me
This section include a form that is compined of The sender name email and the message and a submit button to send the message with an input validation for the email and message content. If any of them is invalide an error message appears in the butom of the invalid field and the submition rejected. If all fields are valid, the message sent and successful message appears to the user.

2. Click Code
3. Click Download ZIP
